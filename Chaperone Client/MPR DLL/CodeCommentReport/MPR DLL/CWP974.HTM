WJ.MPR.WinAPI.RegistryKey.GetValueNames Function
Retrieves an array of strings that contains all the value names associated with this key.

Public string[] GetValueNames ()
  Return Description  
    string An array of strings that contains the value names for the current key.

  
Remarks:
If no value names for the key are found, an empty array is returned. <para>All RegistryKeys are assigned a default value. This is not counted as a value name, and is not returned as part of the result set.</para>
