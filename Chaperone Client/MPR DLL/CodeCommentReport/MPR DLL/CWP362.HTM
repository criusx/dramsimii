WJ.MPR.Reader.ZumaRow.UserData Function
Fill the Row with 16 bits of user data. bit 16 can be used as a WriteFlag. bit 17 is the lock bit for this row.

Public void UserData (byte, byte, bool, bool)
  Type Name Description  
    byte MSB Most Significant 8 bits of the user data in this row.  
    byte LSB Least Significant 8 bits of the user data in this row.  
    bool WriteFlag Can be used to signify that this row has been written.  
    bool LockBit Whether this row is locked.  
  Return Description  
    void
