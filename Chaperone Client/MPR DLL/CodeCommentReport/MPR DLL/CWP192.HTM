WJ.MPR.Reader.ZumaRow Class
A class for a single row of data in a Zuma (Impinj Class 0 Re-writeable) Tag. Each row has 18 bits, 0-17, left justified in three bytes. (only bits 0 and 1 of MSB used) There are 16 rows in a Zuma tag. Row # Function 0 Fab Protect 1 Control Word 2..3 Kill Passcode 4..N EPC Data N+1 EPC CRC N+2..15 User Data EPC length N 64 7 96 9 128 11 Includes methods to initialize a ZumaRow object as the correct type of row.

Access: Public
Base Classes: Object
  Members Description  
    RowNumber Row Number, 0-15, of this row of the Zuma Tag.

 
    Byte0 The MSB, consisting of only 2 bits (17 and 16), of the Row.

 
    Byte1 Bits 15-8 of the Row.

 
    Byte2 Bits 7-0 of the Row.

 
    ToArray Converts the ZumaRow to an array of three bytes.

 
    EPCRow Fills this Row's bits with EPC data.

 
    EPCMSB Interpret the Row as an EPC Row and get the MSB.

 
    EPCLSB Interpret the Row as an EPC Row and get the LSB.

 
    LockBit Interpret the Row as an EPC Row and get the Lock Bit.

 
    PassCodeRow Fill the Row's bits with half a Kill PassCode (12 bits).

 
    FabProtect If Clear == true, fill the Fab Protect bits with 0x0000, else fill it with the Golden Word = 0x997A. Always Row 0.

 
    ControlWord Fill the Row's bits with the proper values for a given EPC length (in bytes).

 
    UserData Fill the Row with 16 bits of user data. bit 16 can be used as a WriteFlag. bit 17 is the lock bit for this row.

 
    ZumaRow Construct a Zuma Row, with just a RowNumber

 
    ZumaRow Construct a Zuma Row with given its 3 bytes of data.

 
    ZumaRow Construct an empty Zuma Row.
