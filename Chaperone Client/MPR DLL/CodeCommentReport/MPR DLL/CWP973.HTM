WJ.MPR.WinAPI.RegistryKey.DeleteValue Function
Deletes the specified value from this key.

Public void DeleteValue (string, bool)
  Type Name Description  
    string name Name of the value to delete.  
    bool throwOnMissingValue Indicates whether an exception should be raised if the specified value cannot be found. If this argument is true and the specified value does not exist then an exception is raised. If this argument is false and the specified value does not exist, then no action is taken
