WJ.MPR.Reader.StatusCode Enum
Command Response Status Codes. These are the Status bytes from Response Frames (or Packets).

Access: Public
  Members Description  
    Error MPR reports an Error.

 
    Complete MPR complete the request. This is the last response frame.

 
    InProgress This is not the last response frame; More frames will follow.

 
    unknown Status Code could not be determined from a response frame.
