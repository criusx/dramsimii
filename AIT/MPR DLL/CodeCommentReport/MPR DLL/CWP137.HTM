WJ.MPR.Reader.Class0ZumaSubcommands Enum
These are the subcommands for Impinj Zuma (Re-writeable Class 0), command 0x18

Access: Public
  Members Description  
    WriteRow Write a Row (18 bits) to a Zuma Tag.

 
    ReadRow Read a Row (18 bits) from a Zuma Tag.

 
    INIT Initialize a Zuma Tag for re-writing. This RF-interface command sets the Fab Protect row to Golden Word.

 
    WriteAck Write a Row (18 bits) to a Zuma Tag, and wait for an acknowledgement. Singulated-only command.
