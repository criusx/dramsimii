WJ.MPR.Reader.ErrorCode Enum
Command Response Error Codes. These Error Codes follow the Status Byte in Frames with a Status Byte = 0xff (StatusCode.Error) <see cref="StatusCode"/>

Access: Public
  Members Description  
    NoError No error reported. This is not received from the MPR, but inferred for Frames with non-Error statuses.

 
    CurrentOutOfRange MPR Current Sense ADC reports out-of-range condition.

 
    VoltageOutOfRange MPR Voltage Sense ADC reports out-of-range condition.

 
    AppHeaderAccessError MPR Could not verify the CRC of the Application Header.

 
    Class0PlusErrorErasingPage Error Programming page (EPC Class 0+ tags only)

 
    Class0PlusErrorProgrammingPage Error Erasing page (EPC Class 0+ tags only)

 
    Class0PlusErrorProgrammingTIB Error Programming Traversal Inhibit Bit (EPC Class 0+ tags only)

 
    Class0PlusErrorLockingPage Error Locking page (EPC Class 0+ tags only)

 
    Class0PlusErrorPageLocked Page cannot be accessed because it is already locked (EPC Class 0+ tags only)

 
    InvalidParams A parameter is out of range for the issued command.

 
    InsufficientData Based on the command issued and the message length, there is not enough data in the message to support the command.

 
    UnsupportedCommand The command is not supported or is unable to be processed by the Reader at this time.

 
    UnsupportedPowerLevel A command has been issued with an unsupported power level.

 
    PLLLockFail On commanding the transmit frequency, the PLL which generates the RF up conversion failed to converge on the required frequency.

 
    AntennaFault The antenna was either not present or shorted.

 
    UnsupportedSubcommand The subcommand issued is not one of the subcommands that can be processed by the reader.

 
    InvalidSubCommandParameter A subcommand parameter is out of range for the issued subcommand.

 
    InvalidNVSTable The Non-Volatile Storage table CRC is invalid.

 
    NVSAlignError The NVS address is not a word address.

 
    NVSProtectionViolation An attempt was made to program(write) or erase a protect address in the flash.

 
    NVSAccessError Indicates an illegal access to the flash during a program(write) or erase commands.

 
    NVSWriteVerifyFail Indicates an unsuccessful write sequence; data written does not match data read back from flash.

 
    IncorrectPasscode The Passcode to enter Maintenance Mode was incorrect.

 
    InvalidCommand The command entered is not supported in the current mode, e.g., maintenance mode commands are only valid while in maintenance mode.

 
    UndefinedError Some other, unspecified error occured.
