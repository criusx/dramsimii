WJ.MPR.Util.ISnapIn Interface
Use this Interface to write a "Snap-In" For the WJ MPR Demo GUIs. Usually this Interface is implented on a Form, and the OnClick EventHandler Shows the Form, which is accessible from the "Snap-In" Menu on the Demo.

Access: Public
  Members Description  
    Init Initialization Routine, that gives the Snap-In a reference to a Reader object.

 
    MenuName The name of the Snap-In as it should appear on the "Snap-Ins" menu of the Demo.

 
    OnOpen The Event Handler to fire when the user selects the SnapIn's menuItem.
