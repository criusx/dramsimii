WJ.MPR.Reader.Inventory Class
A strongly-typed collection of RFIDTag objects.

Access: Public
Base Classes: CollectionBase
  Members Description  
    this Gets or sets the element at the specified index. This property is the indexer for the list.

 
    Add Add a single value to the list.

 
    IndexOf Determines the index of a specific element in the list.

 
    Insert Inserts an element into the list at the specified Index.

 
    Remove Removes the first occurrence of a specific value from the list.

 
    ToArray Convert the list to an array.

 
    Contains Determines whether the Inventory contains a specific RFIDTag.

 
    Contains Determines whether the Inventory contains a RFIDTag with a specific TagID. The TagID of each RFIDTag in the Inventory is compared to val.

 
    Find Determines whether the Inventory contains a RFIDTag with a specific TagID. The TagID of each Tag in the Inventory is compared to val.

 
    Merge Combine two Inventories. If newTags is null, false is returned. Then newTags are added. Lastly Old RFIDTags are Expired. If either adding or expiring changes this Inventory, true is returned.

 
    Add Add an Inventory's RFIDTags to this Inventory. This Inventory is searched for each RFIDTag in newTags, if it is found, its LastRead timestamp is updated. if it is not found, it is added.

 
    persistTime The length of time that a RFIDTag can live in an inventory, without being reread, before it is removed by ExpireTags(). If persistTime == TimeSpan.MaxValue, then RFIDTags are never removed.

 
    ExpireTags The LastRead property of each RFIDTag in this Inventory + the persistTime is compared against the "now" parameter. If the sum is less, then the RFIDTag is removed from this Inventory. If persistTime == TimeSpan.MaxValue, then RFIDTags are never removed.

 
    CountClass Returns the number of Class0 and Class1 RFIDTags in this Inventory.
