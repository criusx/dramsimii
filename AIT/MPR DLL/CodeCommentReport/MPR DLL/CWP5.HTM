WJ.MPR.Reader.RFIDTag Class
Encapsulates an RFID Tag. Includes properties for the Tag ID, the RFID Protocol (enum), "First Read Time" and "Last Read Time" Includes methods for computing Class 0 and Class 1 CRCs. Incomplete: method to convert raw TagID to EPC.
