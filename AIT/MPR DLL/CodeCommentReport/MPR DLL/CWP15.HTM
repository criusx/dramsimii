WJ.MPR.Reader.Inventory.Merge Function
Combine two Inventories. If newTags is null, false is returned. Then newTags are added. Lastly Old RFIDTags are Expired. If either adding or expiring changes this Inventory, true is returned.

Public bool Merge (WJ.MPR.Reader.Inventory)
  Type Name Description  
    Inventory newTags An Inventory containing RFIDTags to add to this Inventory.  
  Return Description  
    bool Whether any new RFIDTags were added or deleted.
