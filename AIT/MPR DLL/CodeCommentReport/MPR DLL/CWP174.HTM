WJ.MPR.Reader.RspStatus Enum
Indicates the possible error sources for an MPRMsg.

Access: Public
  Members Description  
    NoStatus The MPRMsg has no status (yet).

 
    OK No error associated with an MPRMsg.

 
    ErrorRsp An error reported in the status of a response frame.

 
    ErrorComm An error occured and was reported by the communication layer (serial port).

 
    TimeOut A timeout occured = no final response frame received for the MPRMsg.

 
    MsgRcvd A message was recevied, but has not been processed yet.
